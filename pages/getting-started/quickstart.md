# Creating Your First Hypercert

A hypercert is a set of ATProto records on your PDS — any tool that can write ATProto records can create one. These guides cover two common approaches:

- **[Using Code](/getting-started/creating-your-first-hypercert)** — TypeScript and the ATProto API
- **[Using the UI](/getting-started/using-the-scaffold-app)** — a guided wizard, no code required
